# plusコマンド
[![test](https://github.com/Siromi463/robosys2023/actions/workflows/test.yml/badge.svg)](https://github.com/Siromi463/robosys2023/actions/workflows/test.yml)

標準入力から読み込んだ数字をたす。

つかいかた！！！！！！！！！
seq (NUM) | ./plus
上の(NUM)に任意の数値を入力する。

##必要なソフトウェア
* Python
  * テスト済み：3.7 ~ 3.10

##テスト環境
*　Ubuntu
